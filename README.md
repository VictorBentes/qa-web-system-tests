# QA Testing Project

## Sobre
Projeto criado para documentar testes manuais realizados em um sistema web privado, com foco em aprendizado e prática de Quality Assurance (QA).

## Objetivos
- Praticar testes manuais
- Criar casos de teste
- Documentar bugs
- Organizar evidências
- Desenvolver experiência em QA

## Estrutura do Projeto

### test-cases
Casos de teste executados.

### bug-reports
Documentação de bugs encontrados.

### checklists
Listas de validação utilizadas nos testes.

### evidence
Screenshots e evidências dos testes.

## Ferramentas Utilizadas
- GitHub
- Markdown
- Navegador Web
- Windows 11

## Status
Projeto em andamento.
